# sasha-math-bot

Telegram Mini App для репетитора по математике: `index.html` — приложение ученика (профиль, домашние задания, лидерборд, экран «Ещё» с FAQ и приглашением родителя), `teacher.html` — панель учителя (проверка работ, планирование заданий, начисление хуиконов). Общий бэкенд — Supabase.

Организационные данные учеников (группы, дата оплаты, результаты пробников) ведут помощники в Google Sheets; в Supabase они попадают через Apps Script. Подробности архитектуры и спецификация Google Sheets — в `ARCHITECTURE.md` и `GOOGLE_SHEETS_SPEC.md` (локальные файлы, не в git — см. `.gitignore`).

Два Telegram-бота (`main.py` — запускает Mini App для ученика, `parent_bot.py` — читает прогресс ребёнка для родителей и родственников) в этом репозитории не хранятся в git — исключены через `.gitignore`, так как содержат токены. Оба пока запускаются только локально.

Текущий статус реализации и план дальнейшего развития — в `PLAN_OF_IMPLEMENTATION.md` (тоже локальный, не в git). Архитектурный аудит проекта (безопасность, производительность, техдолг) — в `AUDIT.md` (локальный, не в git).
