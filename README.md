# sasha-math-bot

Telegram Mini App для репетитора по математике: `index.html` — приложение ученика (профиль, домашние задания, лидерборд), `teacher.html` — панель учителя (проверка работ, планирование заданий, начисление хуиконов). Общий бэкенд — Supabase.

Организационные данные учеников (группы, дата оплаты, результаты пробников) ведут помощники в Google Sheets; в Supabase они попадают через Apps Script. Подробности архитектуры и спецификация Google Sheets — в `ARCHITECTURE.md` и `GOOGLE_SHEETS_SPEC.md` (локальные файлы, не в git — см. `.gitignore`).

Текущий статус реализации и план дальнейшего развития — в `PLAN_OF_IMPLEMENTATION.md` (тоже локальный, не в git).
